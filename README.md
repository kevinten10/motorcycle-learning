## 1. 摩托车新手学习大纲
- **理论基础**：掌握档位分布（1-N-2-3-4-5-6）、左右手脚分工。
- **静态练习**：熟悉重心、推车、撑大撑（DL250较重，需掌握技巧）。
- **核心技能**：
    - **半联动控制**：所有低速控制的核心。
    - **视线引导**：看哪车去哪，克服目标锁定效应。
    - **渐进式刹车**：前7后3，不要一把捏死。
- **进阶练习**：8字绕桩、紧急避让。

## 2. DL250 专项指南
- **新车磨合**：0-800km (≤5000转)，800-1600km (≤7500转)。
- **保养周期**：
    - **首保**：1000km（机油+机滤）。
    - **机油**：每5000km或一年。
    - **链条**：每500-1000km清洁润滑（DID油封链条）。
- **常用参数**：
    - 机油量：换油约2.1L，连滤芯约2.4L。
    - 胎压：前2.5 / 后2.8 bar。

## 3. 安全驾驶技巧
- **防御性驾驶**：永远假设别人看不见你。
- **盲区意识**：远离大货车，不在并线点停留。
- **雨天注意**：避开井盖和交通线，低抓地力路面减少压弯。

## 4. 装备选购
- **头盔**：全盔（3C/ECE/DOT认证）。
- **手套**：带关节防护。
- **护膝/骑行服**：带内置护甲。
