# Motorcycle Learning & DL250 Tutorial

这是一个围绕豪爵铃木 DL250 整理的新手学习项目。目标不是"快速上手就敢乱骑"，而是让你按顺序建立一套更稳、更安全、更能长期坚持的骑行方法。

![DL250 学习封面](docs/images/dl250-tutorial-cover.svg)

## 这套教程适合谁

- 刚拿到 D/E 证，准备从零开始骑摩托
- 已经会基本起步，但对 DL250 的车重、低速控制、通勤实战还没信心
- 想把"练车、装备、保养、复盘"整理成一条连续学习路径

## 你该怎么用这个项目

1. 先通读本页，建立整体学习框架
2. 按本页的顺序练，每次只练 1-2 个重点
3. 练完后到 [Learning Log](docs/Learning_Log.md) 打卡和复盘
4. 遇到专项问题，再去对应文档深入看

## 第 0 课：先认识 DL250

DL250 很适合新手长期学习，因为它稳定、成熟、容错高，但它并不是"特别轻松的小玩具车"。

你需要先接受 3 个事实：

- 这台车不轻，原地挪车、停车、掉头都比小踏板更讲究动作
- 真正让新手紧张的，通常不是高速，而是低速
- 学习顺序应该是"稳"优先，而不是"快"优先

现阶段你的目标不是压弯，也不是追求速度，而是先做到 5 件事：

- 起步不熄火
- 低速不慌张
- 停车不狼狈
- 通勤有预判
- 出车先检查

---

## 第 1 课：每次出发前先做检查

新手最值得尽早养成的习惯，不是改装，也不是刷视频，而是每次骑之前用 3 分钟看一遍车况。

![出车前检查示意图](docs/images/dl250-pre-ride-check.svg)

出车前至少看 6 项：轮胎、刹车、灯光、油液、链条、车身。

> 详细检查方法和异常排查，见 [DL250 Maintenance](docs/DL250_Maintenance.md)。

---

## 第 2 课：先把最基础的四件事练稳

对于 DL250 新手，优先级最高的 4 个基本动作是：**起步、停车、升降挡、渐进式制动**。

不要急着上复杂路况。只要起步、停车还不稳定，后面的绕桩、掉头、通勤都会放大你的紧张感。

> 具体练习项目、成功标准和打卡方式，见 [Learning Log 第一、二阶段](docs/Learning_Log.md)。

---

## 第 3 课：低速控制是 DL250 学习核心

新手最容易卡住的地方就是 DL250 的低速控制。有效的方法不是"憋住不敢动"，而是：

**半联动 + 稳油门 + 轻后刹**

![低速训练示意图](docs/images/dl250-low-speed-control.svg)

练低速时记住：

- 眼睛看你想去的地方，不要盯地面
- 先求不断脚，再求动作顺
- 车身有角度时，不要突然猛捏前刹
- 低速不稳时，优先想离合和后刹

> 完整的低速训练步骤、绕桩方案和常见错误，见 [DL250 Riding Guide](docs/DL250_Riding_Guide.md)。

---

## 第 4 课：从场地过渡到城市通勤

通勤不是"会拧油门就能上路"，而是你有没有观察节奏和防御意识。

城市通勤最值得练的 4 件事：

- 绿灯起步前先扫左右行人和电动车
- 跟车时给自己留刹车空间
- 并线前先看后视镜再回头确认盲区
- 不在大车旁边长时间并行

如果你第一次开始真实通勤，建议：先选熟悉路线 → 先避开高峰 → 先白天再夜间 → 先短距离再长距离。

> 通勤实战细节、雨天/夜间骑行、长途准备，见 [DL250 Riding Guide](docs/DL250_Riding_Guide.md)。

## 第 5 课：装备不是面子，是学习成本控制

对 DL250 新手来说，最常见的问题不是"高速飞出去"，而是低速倒车、窄路掉头慌张、雨天打滑。所以装备优先防护，优先覆盖最容易出现事故的场景。

一句话概括：**先把头、手、脚和躯干的基础防护补齐，再考虑相机、对讲、改装件。**

> 按必买/建议买/选买整理的完整清单和预算分配，见 [DL250 Gear Guide](docs/DL250_Gear_Guide.md)。

---

## 第 6 课：把保养变成骑行的一部分

DL250 不难养，但新手很容易只会骑、不会看车况。

你至少要记住这些高频项目：

| 项目 | 周期 |
| --- | --- |
| 首保（机油+机滤） | 1000 km |
| 机油更换 | 每 5000 km 或一年 |
| 链条清洁润滑 | 每 500-1000 km |
| 胎压检查 | 每周 |
| 冷却液更换 | 约两年 |

如果你能做到"每周一次检查 + 每次骑前快速确认"，你对车的掌控感会明显提升。

> 完整保养参数、出车前 3 分钟检查清单和异常排查，见 [DL250 Maintenance](docs/DL250_Maintenance.md)。

## 一个适合新手的 30 天学习节奏

### 第 1 周
- 熟悉按键、边撑、熄火开关、油箱盖、大撑
- 原地推车，适应车重
- 只练起步、停车、半联动

### 第 2 周
- 加入升降挡和渐进式制动
- 开始大间距绕桩和宽 U 型掉头

### 第 3 周
- 加入窄路掉头、坡道起步
- 选择熟悉路段做短距离真实通勤

### 第 4 周
- 复盘通勤中的路口、跟车、并线和盲区问题
- 固定做出车前检查、洗车和链条保养

## 学习时只抓这几个原则

- 每次只练 1-2 个目标
- 先练慢，再练顺，最后才是复杂场景
- 练不好时先减难度，不要硬顶
- 出现紧张和手忙脚乱，先回到基础动作
- 所有进步都记录到 [Learning Log](docs/Learning_Log.md)
