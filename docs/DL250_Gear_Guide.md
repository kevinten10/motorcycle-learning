# DL250 新手装备调研与购买建议

本指南是 [README 教程](../README.md) 的配套装备手册，重点解决"买什么、怎么选、花多少"的问题。

## 核心原则

**先防护，后舒适，最后才是个性化和装饰。**

DL250 新手最常见的问题不是"高速飞出去"，而是低速倒车、原地挪车失误、窄路掉头慌张、雨天打滑。所以装备优先覆盖最容易出事故的场景。

## 选购要点

### 头盔

- **认证标准**：3C（国内最低要求）→ DOT → ECE 22.06（推荐）→ SNELL（最高）
- **材质**：ABS（入门）→ 玻璃纤维（中端）→ 碳纤维（轻量高端）
- **试戴**：戴 5 分钟不夹太阳穴、不压额头，摇头不移位
- **更换周期**：3-5 年或经历过一次撞击后立即更换
- **不推荐**：半盔和揭面盔（防护面积不够）

### 手套

- **夏季**：网眼透气款，带硬质指关节保护
- **冬季**：防风保暖款，最好有触屏指尖
- **必须**：掌根有缓冲垫，指关节有硬壳或滑动块
- **不推荐**：半指手套、普通皮手套

### 骑行靴

- **高度**：必须覆盖脚踝，低帮运动鞋不够
- **鞋底**：防滑、耐磨，换挡部位有加强
- **日常兼容**：短款骑行靴可以日常穿
- **不推荐**：帆布鞋、凉鞋、普通运动鞋

### 骑行服

- **护甲位置**：肩、肘、背（背部护甲容易被忽略，但很重要）
- **夏季**：网眼透气款，避免中暑
- **四季**：可拆卸内胆的拉力服或多功能款
- **日常**：牛仔骑行裤、连体骑行内衣可以兼顾通勤外观

### 雨衣

- **分体式**：上衣 + 雨裤，不会被风兜住，更安全
- **一体式**：穿脱方便，但高速时容易被风灌入
- **建议**：通勤选分体式，摩旅选带反光条的

## 预算分配建议

| 预算总额 | 分配建议 |
| --- | --- |
| 1000 元以内 | 全盔 600 + 手套 100 + 反光背心 50 → 剩余存着买骑行靴 |
| 2000-3000 元 | 全盔 1000 + 手套 200 + 骑行靴 500 + 骑行服 800 |
| 5000 元以上 | 以上全部 + 骑行裤 + 雨衣 + 冬季手套 |

**不要省的**：头盔和手套。这两样是发生事故时最先着地的部位。

## DL250 加装件建议

| 加装件 | 必要性 | 说明 |
| --- | --- | --- |
| 护杠 | **强烈建议** | DL250 满油很重，原地倒车时保护车壳和腿 |
| 手机支架 | 建议 | 导航用，选减震款保护手机摄像头 |
| 尾箱 | 按需 | 通勤放头盔和杂物，但会增加车重和风阻 |
| 边箱 | 按需 | 摩旅时有用，但会显著增加车宽 |
| 射灯 | 夜间通勤建议 | 提高被看见的概率，注意法规限制 |

> 加装件会改变车重和重心，加装后需要重新适应起步、制动和掉头。
