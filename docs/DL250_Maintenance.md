# 豪爵铃木 DL250 保养手册要点

本手册是 [README 教程](../README.md) 的配套参数手册，重点提供保养数据和检查清单。

## 0. 日常使用定位

- **适合场景**：城市通勤、短中途摩旅、新手进阶练习
- **车辆特点**：稳定、耐用、容错高，但车重不轻，低速和原地挪车要更讲方法
- **保养核心原则**：少暴力、勤检查、按周期更换耗材

## 1. 润滑系统

| 项目 | 规格 |
| --- | --- |
| 推荐机油 | 10W-40 / 10W-50（SG 级以上，推荐 SJ/SL） |
| 更换周期 | 每 5000 km 或一年（以先到为准） |
| 首保 | 1000 km（机油 + 机滤） |
| 换油量（不含滤芯） | 约 2.1 L |
| 换油量（含滤芯） | 约 2.4 L |
| 推荐品牌 | 豪爵原厂、壳牌爱德王子、摩特 7100 |

## 2. 链条

| 项目 | 规格 |
| --- | --- |
| 类型 | DID O-Ring 油封链条 |
| 清洁周期 | 每 500-1000 km |
| 润滑周期 | 每 500-1000 km（清洁后润滑） |
| 清洁剂 | 煤油或专用链条清洗剂（严禁汽油或钢丝刷） |
| 松紧标准 | 上下波动约 20-30 mm |

**检查重点**：
- 链条松紧是否明显变大
- 润滑后是否仍有明显异响
- 链条与牙盘是否有异常磨损

## 5. 出车前 3 分钟检查

这是新手最值得尽早养成的习惯。每次骑之前花 3 分钟过一遍：

| 检查项 | 看什么 |
| --- | --- |
| 轮胎 | 胎压、胎纹、异物、鼓包 |
| 刹车 | 前后刹手感是否正常 |
| 灯光 | 近光、远光、转向灯、刹车灯 |
| 油液 | 油量、冷却液液位、地面渗漏 |
| 链条 | 是否过松、过脏、缺油 |
| 车身 | 后视镜角度、边撑回位、护杠/尾箱固定 |

## 6. 新手最容易忽略的点

- **低速时不要捏死前刹**：DL250 车重较大，车把有角度时猛捏前刹更容易倒车
- **长时间不骑也要看胎压**：胎压不足会让车更笨重，转向和刹车都受影响
- **加装件越多越要适应**：护杠、尾箱、边箱会改变车宽和重心
- **洗车后不要忘了链条**：洗完车如果链条没重新润滑，异响和磨损都会加快

## 7. 常见异常与排查方向

| 现象 | 先查什么 |
| --- | --- |
| 起步闯动明显 | 离合控制 → 是否拖挡 → 链条是否过松 |
| 刹车手感突然变差 | 刹车片磨损 → 刹车油状态 |
| 车身跑偏或发飘 | 胎压 → 轮胎磨损 → 负载是否均匀 |
| 怠速或散热异常 | 冷却液液位 → 风扇是否正常介入 |
| 链条异响 | 链条松紧 → 是否缺油 → 牙盘磨损 |

## 8. 常见问题

- **原地倒车**：DL250 满油满箱时很沉，一定要装护杠
- **车重感**：掉头时控制好重心，慢速转弯不要猛捏前刹
- **大撑难上**：双手稳住车把和后扶手，借助脚踩大撑支点发力，不要硬拽
