# 摩托车驾驶学习日志 (DL250 专项)

本日志用于记录从零开始学习摩托车驾驶的过程，配合 [README 教程](../README.md) 使用。

## 使用方法

- 每次练习只盯 1-2 个目标，不要一口气练太多项目
- 每项练习尽量写下"成功标准"，例如"20 次起步 0 熄火"
- DL250 车重偏大，低速训练优先追求稳定，不追求动作好看
- 建议记录格式：`日期 - 练习科目 - 完成度 - 体会/不足 - 下次目标`

## 第一阶段：理论武装与装备准备

**目标**：上路之前把证照、装备和车辆基本操作全部到位。

- [ ] **1.1 法律法规**：D/E 证状态：(未开始/进行中/已考取)
- [ ] **1.2 装备配置**：
    - [ ] 头盔：
    - [ ] 手套：
    - [ ] 骑行靴/骑行服：
    - [ ] > 装备选购建议见 [DL250 Gear Guide](DL250_Gear_Guide.md)
- [ ] **1.3 车辆熟悉**：
    - [ ] 熟悉所有按键功能
    - [ ] 尝试支起/放下大撑（DL250 较重，需技巧）
    - [ ] 原地推车前后各 10 米，适应车重
    - [ ] 熟悉驻车、熄火开关、油箱盖、边撑熄火逻辑

**完成标准**：能独立完成原地推车和所有按键操作，装备已到位。

---

## 第二阶段：封闭场地基础操纵

**目标**：起步 0 熄火、停车 0 慌张、升降挡顺畅、制动有节奏。

- [ ] **2.1 静态推车**：绕 8 字推车，感受重心
- [ ] **2.2 起步与停止**：
    - [ ] 1 档平稳起步 20 次（目标：0 熄火）
    - [ ] 1 档慢速滑行，控制离合半联动
    - [ ] 起步时做到"先稳离合，再轻补油"
    - [ ] 低速停车时养成"车正再落脚"的习惯
- [ ] **2.3 换挡练习**：
    - [ ] 1-2-3 档升降档顺滑度
    - [ ] 寻找空档（N 档）的脚感
    - [ ] 学会减速时提前降挡，避免拖挡闯车
- [ ] **2.4 制动练习**：
    - [ ] 20km/h 紧急制动（感受前/后刹分配）
    - [ ] 30km/h 直线制动，练"先轻后重"渐进式刹车
    - [ ] 低速带一点后刹保持车身稳定

**完成标准**：连续起步 20 次 0 熄火，升降挡无明显顿挫，直线制动车身不偏。

---

## 第三阶段：进阶驾驶技巧

**目标**：低速绕桩不断脚、窄路掉头不慌、有基本的紧急避障能力。

> 详细训练方法和常见错误，见 [DL250 Riding Guide](DL250_Riding_Guide.md)。

- [ ] **3.1 转向技巧**：
    - [ ] 视线引导（看向弯心）
    - [ ] 推舵练习（Counter-steering）
- [ ] **3.2 慢速控制**：
    - [ ] 窄路掉头
    - [ ] 慢速绕桩（不落脚）
    - [ ] 半联动 + 轻后刹 + 稳油门组合练习
    - [ ] 掉头时避免盯地面和车头
- [ ] **3.3 紧急避障**：
    - [ ] 快速闪躲障碍物
    - [ ] 先减速再绕，不把避障练成"盲目猛打把"

**完成标准**：大间距绕桩不断脚，窄路掉头 3 次成功 2 次，紧急制动车身不失控。

---

## 第四阶段：城市通勤实战

**目标**：能在熟悉路段独立完成通勤，有基本的防御意识。

> 通勤实战细节和预判技巧，见 [DL250 Riding Guide 城市通勤篇](DL250_Riding_Guide.md)。

- [ ] **4.1 基本通勤**：
    - [ ] 选择熟悉路段、避开高峰
    - [ ] 红绿灯起步时观察左右来车和行人
    - [ ] 给自己留出可撤退空间，不贴大车、不钻盲区
    - [ ] 复杂路口预判
- [ ] **4.2 车辆自主维护**：
    - [ ] 第一次洗车与链条保养
    - [ ] 检查胎压与液位
    - [ ] 学会识别链条过松、胎压不足、刹车手感异常
    - [ ] > 保养参数和检查清单见 [DL250 Maintenance](DL250_Maintenance.md)

**完成标准**：连续 5 天独立完成同一条通勤路线，无险情。

---

## 第五阶段：DL250 专项提升

**目标**：适应更多场景（坡道、载重、长途、天气），建立长期骑行习惯。

> 各场景的详细建议，见 [DL250 Riding Guide](DL250_Riding_Guide.md) 和 [DL250 Maintenance](DL250_Maintenance.md)。

- [ ] **5.1 坡道与载重**：
    - [ ] 坡道起步不溜车
    - [ ] 带尾箱/带物品后重新适应刹车距离
- [ ] **5.2 长时间骑行**：
    - [ ] 连续骑行 1 小时后复盘坐姿、手腕和肩颈疲劳
    - [ ] 熟悉加油、休息、补水节奏
- [ ] **5.3 雨天与夜间**：
    - [ ] 雨天降低动作幅度，避开井盖、标线和白色箭头
    - [ ] 夜间提前减速、远离强光盲区

**完成标准**：能独立完成一次 2 小时以上的短途骑行，涵盖至少 2 种场景。
