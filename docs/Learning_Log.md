# 摩托车驾驶学习日志 (DL250 专项)

本日志用于记录从零开始学习摩托车驾驶的过程。

## 使用方法
- 每次练习只盯 1-2 个目标，不要一口气练太多项目。
- 每项练习尽量写下“成功标准”，例如“20 次起步 0 熄火”。
- DL250 车重偏大，低速训练优先追求稳定，不追求动作好看。
- 建议记录格式：`日期 - 练习科目 - 完成度 - 体会/不足 - 下次目标`

## 第二阶段：封闭场地基础操纵
- [ ] **2.1 静态推车**：绕 8 字推车，感受重心。
- [ ] **2.2 起步与停止**：
    - [ ] 1 档平稳起步 20 次 (目标：0 熄火)
    - [ ] 1 档慢速滑行，控制离合半联动
    - [ ] 起步时做到“先稳离合，再轻补油”
    - [ ] 低速停车时养成“车正再落脚”的习惯
- [ ] **2.3 换挡练习**：
    - [ ] 1-2-3 档升降档顺滑度
    - [ ] 寻找空档 (N 档) 的脚感
    - [ ] 学会减速时提前降挡，避免拖挡闯车
- [ ] **2.4 制动练习**：
    - [ ] 20km/h 紧急制动 (感受前/后刹分配)
    - [ ] 30km/h 直线制动，练“先轻后重”渐进式刹车
    - [ ] 低速带一点后刹保持车身稳定

## 第三阶段：进阶驾驶技巧
- [ ] **3.1 转向技巧**：
    - [ ] 视线引导 (看向弯心)
    - [ ] 推舵练习 (Counter-steering)
- [ ] **3.2 慢速控制**：
    - [ ] 窄路掉头
    - [ ] 慢速绕桩 (不落脚)
    - [ ] 半联动 + 轻后刹 + 稳油门组合练习
    - [ ] 掉头时避免盯地面和车头
- [ ] **3.3 紧急避障**：
    - [ ] 快速闪躲障碍物
    - [ ] 先减速再绕，不把避障练成“盲目猛打把”

## 第四阶段：实战与维护
- [ ] **4.1 城市通勤实战**：
    - [ ] 复杂的路口预判
    - [ ] 防御性驾驶心态
    - [ ] 会在红绿灯起步时观察左右来车和行人
    - [ ] 会给自己留出可撤退空间，不贴大车、不钻盲区
- [ ] **4.2 车辆自主维护**：
    - [ ] 第一次洗车与链条保养
    - [ ] 检查胎压与液位
    - [ ] 学会识别链条过松、胎压不足、刹车手感异常
    - [ ] 骑行前完成 3 分钟快速检查

## 第五阶段：DL250 专项提升
- [ ] **5.1 坡道与载重**：
    - [ ] 坡道起步不溜车
    - [ ] 带尾箱/带物品后重新适应刹车距离
- [ ] **5.2 长时间骑行**：
    - [ ] 连续骑行 1 小时后复盘坐姿、手腕和肩颈疲劳
    - [ ] 熟悉加油、休息、补水节奏
- [ ] **5.3 雨天与夜间**：
    - [ ] 雨天降低动作幅度，避免井盖、标线和白色箭头
    - [ ] 夜间学会提前减速、远离强光盲区
