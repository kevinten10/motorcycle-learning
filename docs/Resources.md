# 摩托车驾驶学习资料清单

这份清单汇集了摩托车骑行中最核心的理论、视频教程和社区资源。

## 1. 理论学习 (核心)
- **《摩托车驾驶圣经》(Twist of the Wrist II)**:
    - 简介：全球骑行界的“圣经”，深入浅出地讲解了弯道技巧、身体姿势、推舵原理。
    - 关键词搜索：B站搜索“摩托车驾驶圣经”。
- **防御性驾驶 (Defensive Riding)**:
    - 简介：在复杂的城市交通中，预判其他司机的错误，始终为自己留出“逃生空间”。
    - 推荐UP主：九段骑行、威风堂摩托车。

## 2. 视频教程 (实操建议)
- **九段骑行 - 《新手进阶系列》**:
    - 特点：非常硬核，适合从零开始，讲解离合器控制、小半径掉头、紧急避障。
- **威风堂摩托车 - 《摩托车安全驾驶教程》**:
    - 特点：更偏向安全与意识，涵盖了各种路况下的应对策略。
- **大排君 - 《新手课堂》**:
    - 特点：讲解装备选择、买车注意事项、基础维护。

## 3. DL250 专项资料
- **豪爵铃木官网**:
    - 作用：下载最新的电子版用户手册。
- **B站搜索 "DL250 保养"**:
    - 重点关注：链条保养、机油机滤更换流程、护杠安装建议。

## 4. 社区与工具
- **摩托范/哈罗摩托 (App)**:
    - 作用：查询车辆参数、保养周期、当地摩友交流。
- **驾考宝典**:
    - 作用：备考 D/E 证的理论部分。

---
*建议：先看《驾驶圣经》建立理论框架，再去场地对照九段骑行的视频进行实操。*
